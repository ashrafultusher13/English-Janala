# English Janala

## Overview
English Janala is an interactive platform for learning English with exercises on vocabulary, phrases, and grammar.

## Technology Stack
- React.js
- Tailwind CSS
- JavaScript (ES6+)

## Main Features
- Vocabulary practice with quizzes
- Grammar exercises
- Phrase learning with examples
- Responsive UI for desktop & mobile

## Dependencies
- react-icons
- react-router-dom
- react-toastify

## Setup & Run Locally
1. Clone the repo: `git clone [repo-link]`
2. Install dependencies: `npm install`
3. Start the app: `npm start`
